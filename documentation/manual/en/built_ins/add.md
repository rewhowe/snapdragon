## `被加数に 加数を 足す`, `加数を 足す`

Adds `加数` to `被加数`. If `被加数` is omitted: adds `加数` to `それ`.

| Parameters                         | Return                         | ひらがな Allowed? |
| ---------------------------------- | ------------------------------ | ----------------- |
| `被加数`: Number<br>`加数`: Number | The sum of `加数` and `被加数` | Yes               |

Example:

```
1に 2を 足す ※ 3
3を 足す     ※ 6
```
