## `被除数を 除数で 割る`, `除数で 割る`

Divides `被除数` by `除数`. If `被除数` is omitted: Divides `それ` by `除数`.

| Parameters                          | Return                              | ひらがな Allowed? |
| ----------------------------------- | ----------------------------------- | ----------------- |
| `被除数`: Number<br>`除数`: Number  | The dividend of `被除数` and `除数` | Yes               |

Example:

```
42を 7で 割る ※ 6
2で 割る      ※ 3
```
