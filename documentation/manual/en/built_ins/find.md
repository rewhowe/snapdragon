## `対象列で 要素を 探す`

Returns the corresponding key or index of the first instance of `要素` if found within `対象列`. Returns `無` if not found.

| Parameters                                    | Return           | ひらがな Allowed? |
| --------------------------------------------- | ---------------- | ----------------- |
| `対象列`: Array or String<br>`要素`: Anything | String or Number | Yes               |

Example:

```
ホゲは 「あかさたなはまやらわをん」

ホゲで 「は」を 探す ※ 5
```
