## `被乗数に 乗数を 掛ける`, `乗数を 掛ける`

Multiplies `被乗数` by `乗数`. If `被乗数` is omitted: Multiplies `それ` by `乗数`.

| Parameters                          | Return                             | ひらがな Allowed? |
| ----------------------------------- | ---------------------------------- | ----------------- |
| `被乗数`: Number<br>`乗数`: Number  | The product of `被乗数` and `乗数` | Yes               |

Example:

```
60に 60を 掛ける ※ 3600
24を 掛ける      ※ 86400
```
