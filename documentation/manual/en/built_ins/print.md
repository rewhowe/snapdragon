## `言葉と 言う`, `言葉を 言う`

Prints `言葉` to stdout. `言葉を 言う` differs in semantics only.

| Parameters     | Return | ひらがな Allowed? |
| -------------- | ------ | ----------------- |
| `言葉`: String | `言葉` | Yes               |

Example:

```
「こんにちは、世界」と 言う  ※ こんにちは、世界

それを 言う                  ※ こんにちは、世界
```
