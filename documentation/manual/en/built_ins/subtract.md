## `被減数から 減数を 引く`, `減数を 引く`

Subtracts `減数` from `被減数`. If `被減数` is omitted: Subtracts `減数` from `それ`.

| Parameters                         | Return                                | ひらがな Allowed? |
| ---------------------------------- | ------------------------------------- | ----------------- |
| `被減数`: Number<br>`減数`: Number | The difference of `減数` and `被減数` | Yes               |

Example:

```
10から 2を 引く ※ 8
5を 引く        ※ 3
```
