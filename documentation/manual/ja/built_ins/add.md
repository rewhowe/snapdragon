## `被加数に 加数を 足す`, `加数を 足す`

加法。`被加数`を略すと`それ`に足す。

| 引数                           | 戻り値                 | ひらがな可能 |
| ------------------------------ | ---------------------- | ------------ |
| `被加数`: 数値<br>`加数`: 数値 | `加数`と`被加数`の加算 | ○           |

例）

```
1に 2を 足す ※ 3
3を 足す     ※ 6
```
