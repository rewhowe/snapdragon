## `データを ポイ捨てる`

デバグのコマンドラインオプションが有効の場合：`データ`を標準出力に送信する。ビックリマーク（全角・半角可能）を付けると実行が停止する。

| 引数             | 戻り値   | ひらがな可能 |
| ---------------- | -------- | ------------ |
| `データ`: 型自由 | `データ` | ☓           |

例）

```
ホゲは 配列
ホゲの 「フガ」は 「ピヨ」
ホゲを ポイ捨てる         ※ {"フガ": "ピヨ"} (デバッグ有効時のみ)
```
