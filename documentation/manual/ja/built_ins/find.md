## `対象列で 要素を 探す`

`対象列`の最初の`要素`の該当するキーを返す。見つからない場合、`無`を返す。

| 引数                                       | 戻り値         | ひらがな可能 |
| ------------------------------------------ | -------------- | ------------ |
| `対象列`: 配列 or 文字列<br>`要素`: 型自由 | 文字列 or 数値 | 〇           |

例）

```
ホゲは 「あかさたなはまやらわをん」

ホゲで 「は」を 探す ※ 5
```
