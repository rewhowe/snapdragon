## `被除数を 除数で 割った余りを求める`, `除数で 割った余りを求める`

剰余算。`被除数`を略すと`それ`を割った余りを求める。

| 引数                           | 戻り値                       | ひらがな可能 |
| ------------------------------ | ---------------------------- | ------------ |
| `被除数`: 数値<br>`除数`: 数値 | `被除数`を`除数`で割った余り | `わった余りを求める`,<br>`わったあまりを求める`,<br>or `わったあまりをもとめる` |

例）

```
30を 11で 割った余りを求める ※ 8
3で 割った余りを求める       ※ 2
```
