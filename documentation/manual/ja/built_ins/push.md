## `対象列に 要素を 押し込む`, `対象列に 要素を 追加する`

`要素`を`対象列`の末尾（最高指数）に押し込む。`対象列`が文字列の場合：`要素`は文字列に限る。

`追加する`はエイリアスだ。

`対象列`が変更される。

| 引数                                         | 戻り値   | ひらがな可能   |
| -------------------------------------------- | -------- | -------------- |
| `対象列`: 配列または文字列<br>`要素`: 型自由 | `対象列` | `おしこむ`のみ |

例）

```
ホゲは 「あ」、「い」、「う」 ※  {0: "あ", 1: "い", 2: "う"}

ホゲに 「え」を 押し込む      ※ {0: "あ", 1: "い", 2: "う", 3: "え"}
ホゲに 「お」を 追加する      ※ {0: "あ", 1: "い", 2: "う", 3: "え", 4: "お"}
```
