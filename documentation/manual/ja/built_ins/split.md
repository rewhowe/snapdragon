## `対象列を 区切りで 分割する`

`対象列`を`区切り`で分けて、それぞれの部分を新しい配列で返す。

`対象列`が配列の場合：配列を持つ配列を返す。

`対象列`が文字列の場合：文字列の配列を返す。`区切り`が文字列。

| 引数                                         | 戻り値         | ひらがな可能 |
| -------------------------------------------- | -------------- | ------------ |
| `対象列`: 配列 or 文字列<br>`区切り`: 型自由 | 配列 or 文字列 | ☓           |

例）

```
挨拶は 「こんにちは、世界」

挨拶を 6から 8まで 切り抜く ※ 世界
挨拶を 表示する             ※ こんにちは、
```
