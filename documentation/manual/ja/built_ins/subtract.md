## `被減数から 減数を 引く`, `減数を 引く`

減法。`被減数`を略すと`それ`から引く。

| 引数                           | 戻り値                 | ひらがな可能 |
| ------------------------------ | ---------------------- | ------------ |
| `被減数`: 数値<br>`減数`: 数値 | `減数`と`被減数`の減算 | ○           |

例）

```
10から 2を 引く ※ 8
5を 引く        ※ 3
```
