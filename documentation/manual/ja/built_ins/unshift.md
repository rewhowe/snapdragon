## `対象列に 要素を 先頭から押し込む`

`要素`を`対象列`の先頭（最低指数）に押し込む。`対象列`が文字列の場合：`要素`は文字列に限る。

`対象列`が変更される。

| 引数                                         | 戻り値   | ひらがな可能           |
| -------------------------------------------- | -------- | ---------------------- |
| `対象列`: 配列または文字列<br>`要素`: 型自由 | `対象列` | `先頭からおしこむ`のみ |

例）

```
ホゲは 「あ」、「い」、「う」    ※ {0: "あ", 1: "い", 2: "う"}

ホゲに 「え」を 先頭から押し込む ※ {0: "え", 1: "あ", 2: "い", 3: "う"}
```
